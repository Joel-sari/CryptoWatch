## CryptoWatch

Real-time cryptocurrency tracker using the CoinGecko API. Built with React and styled-components.

### Features
- Live price updates
- Interactive charts
- Responsive UI

### Tech Stack
- ReactJS, JavaScript, CoinGecko API, Styled-Components
